Employees

Employees resource

  • GET : Bookings-All employees have booked their rooms
  • GET : Bookings-Show all the employees who haven't been allocated with the rooms yet(Admin)
  • GET : FAIL Get employee due to Empty/Different domain email
  • GET : Fail Get employee due to Email Not Found
  • GET : Get employee details using email
  • GET : Get list of employees whom room is not allocated(User)
  • POST : New Employee-Employee already exits
  • POST : New Employee-Employee created successfully
  • POST : New Employee-Gender Details : other than (M/F)
  • POST : New Employee-Use Josh software email
  • POST : New Employees-Wrong parameters passed (blank emp_id/only firstname)
  • PUT : Check if there are exact 3 employees for room booking
  • PUT : Successful booking for all the three employees
  • PUT : When one of the employee is already allocated
